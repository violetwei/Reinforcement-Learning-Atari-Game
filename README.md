# Reinforcement-Learning-Atari-Game
Train an agent to play Atari games using Reinforcement Learning Algorithms (e.g. Deep Q networks, A3C, etc.)

Reinforcement Learning (RL) enables computers/robots to successfully plays games (at human-level) using only pixel inputs

Atari grames [list](https://gym.openai.com/envs/#atari)

### Environment
- Install [VirtualBox](https://www.virtualbox.org/) to get Linux working. 

- Once it's installed, install a virtualized instance of [Ubuntu](https://ubuntu.com/download/desktop)

Run VirtualBox and create a new VM with the NEW option. Select Linux & Ubuntu 64 bit in options.

- Install Stella

Stella is the emulator for playing the Atari ROM files

Navigate to Firefox in VM, Download here: https://stella-emu.github.io/downloads.html

- Install PyCharm

- Install Gym

Gym is the environment for testing the machine learning algorithms

```
sudo apt install python-pip

pip install gym
```
